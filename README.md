# do.did

**Simplest todo list for your terminal.**

## Installation

```
npm install do.did --global
```

## Quickstart

```diff
$ do

$ do this
+ this

$ do that
• this
+ that

$ do
• this
• that

$ did that
• this
- that

$ did this
- this

```
