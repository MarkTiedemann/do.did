# do.did

**Simplest todo list for your terminal.**

## Installation

```
npm install do.did --global
```

## Quickstart

```
$ do

$ do this
+ this [green]

$ do that
- this
+ that [green]

$ do
- this
- that

$ did that
- this
- that [red]

$ did this
- this [red]

```
